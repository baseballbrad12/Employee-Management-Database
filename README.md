# Employee Management Database
  [![License: MIT](https://img.shields.io/badge/license-MIT-red)]
  ---------------------------------------
  ## Description
  This is a CLI application that uses node, inquirer, and MySQL to help companies manage their employees digitally.
  ---------------------------------------
  ## Installation
  Please install node, inquirer and MySQL. Instructions for the installation of those extensions can be found on the internet.
  ---------------------------------------
  ## Usage
  This application can be initiated in the root directory by running a node index.js. The user is then prompted with questions necessary to incorporate employee data. The employee data is then connected to queries to be displayed in the MySQL database.
